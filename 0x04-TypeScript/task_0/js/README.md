# 0x04. TypeScript

## About the Project
This project introduces the basics of TypeScript and how it’s used in frontend development.  
It focuses on understanding types, interfaces, classes, functions, and how to work with the DOM using TypeScript.  
The project setup also includes Webpack, Jest, and ESLint for proper development workflow and testing.

---

## Learning Objectives
At the end of this project, you should be able to:

- Use basic types in TypeScript  
- Define and use interfaces and classes  
- Work with the DOM in TypeScript  
- Use generic types and namespaces  
- Merge declarations and use ambient namespaces  
- Understand basic nominal typing
